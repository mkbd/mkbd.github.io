
## oauth2Api(Mashup)공통앱

## 소개
- oauth2 방식으로 공개된 api 많음
- twitter, facebook, tistory, whooing, 다음지도, 비트코인 등 다수 존재
- 이를 아우루는 공통앱을 생성.
- 매시업을 쉽게 할 수 있도록 템플릿화 한다
- 공통된 부분을 정리한다
- 여러앱을 함쳐놓은 것 같이 사용 가능 하다.
